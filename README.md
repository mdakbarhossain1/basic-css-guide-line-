# Design System

This is a **responsive design system** built with **vanilla CSS**. It provides a **master CSS file**, reusable components, and utility classes for consistent UI/UX across projects.

## Features

- Responsive typography
- Buttons: default, primary, secondary, outline
- Cards with shadows and border-radius
- Forms: inputs, textareas, selects
- Grid system: `.row` and `.col-*`
- Fully responsive for mobile, tablet, desktop, extra-wide screens
